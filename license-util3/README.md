# Program to Activate License on a CPD Trial Cluster

## Compile
* On MacOSX, run:
```
cd lutil3 && go build
```
* To compile for a Linux distribution, download dependencies using `dep init` and use the build script:
```
cd lutil3 && dep init
cd build && ./build.sh lutil3
```
## Usage
* Run binary:
```
./lutil3
```
* Read and accept license.
